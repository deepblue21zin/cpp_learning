1. <algorithm> 헤더와 max() 함수

C++ 표준 라이브러리인 <algorithm>은 이름 그대로 알고리즘에 유용한 함수들을 담고 있습니다. 그중 max 함수는 값을 비교할 때 매우 유용합니다.

    2개 수 비교: max(a, b) 형태로 사용하며, 둘 중 큰 값을 반환합니다.

    3개 이상의 수 비교: 작성하신 코드처럼 **중괄호 { }**를 사용하여 max({a, b, c}) 형태로 작성하면 여러 개 중 가장 큰 값을 찾아줍니다. (이는 C++11 버전부터 지원되는 '초기화 리스트'를 활용한 방식입니다.)

2. 조건문의 우선순위와 논리 연산자

이 문제의 핵심은 **"가장 까다로운 조건부터 먼저 걸러내는 것"**입니다.

    && (AND): 모두 참이어야 실행됩니다. 3개의 눈이 모두 같을 때(a==b && b==c)를 가장 먼저 체크하여 가장 큰 상금을 먼저 처리한 점이 좋습니다.

    || (OR): 하나만 참이어도 실행됩니다. 2개의 눈이 같은 경우를 찾을 때 유용합니다.

    else if의 흐름: 1. 먼저 3개가 다 같은지 확인. 2. 아니라면, 2개만 같은지 확인. 3. 그것도 아니라면, 모두 다른지 확인. 이 순서가 바뀌면(예를 들어 2개만 같은지 먼저 확인하면) 3개가 같은 경우도 2개만 같다고 판정될 수 있는데, 코드를 아주 정석적으로 잘 짜셨습니다.

3. 중복된 조건의 최적화 (팁)

작성하신 코드에서 else if(a==b || a==c)와 그 아래의 조건들이 조금 겹치는 부분이 있어요. 이를 더 깔끔하게 정리하면 다음과 같이 생각할 수 있습니다.
상황	비교 로직	상금 계산에 쓰는 변수
3개 일치	a == b && b == c	a (아무거나)
2개 일치 (1)	a == b 또는 a == c	a
2개 일치 (2)	b == c	b
모두 다름	그 외 (else)	max({a, b, c})

    알아두면 좋은 점: a==b || a==c에서 참이 아니어서 다음 else if로 넘어왔다면, 이미 a는 b나 c와 같지 않다는 뜻입니다. 따라서 마지막 2개 일치 조건은 else if(b == c) 하나만으로도 충분하게 됩니다.
