1	

실행 안 됨

while(N=0)
	N=0은 대입 연산이라 결과가 0(거짓)이 되어 반복문이 아예 실행되지 않음.	
    while(true)로 변경하고 내부에서 if(N==0) break; 사용	
    조건 검사 전에 입력을 먼저 받아야 올바르게 종료 여부를 판단할 수 있음.
2	

컴파일 경고

'N' is used uninitialized
	int N; 선언 후 **값이 없는 상태(쓰레기값)에서 while(N==0) 비교를 시도함.	
    while 조건에서 N을 빼고, 무한 루프 내부에서 입력을 받음.	
    변수는 반드시 값이 들어간 후에 사용(비교)해야 안전함.
3	

무한 루프

(프로그램이 멈춤)
자릿수를 세는 while(N>0) 안에서 N의 값을 줄여주는 코드가 없었음.	

반복문 마지막에

N = N / 10;** 추가
숫자를 10으로 나눠서 0이 되게 만들어야 반복문이 끝남.
4	

1회만 실행되고 종료

(계속 입력 안 됨)
	**return 0;**이 while 반복문 안쪽에 있었음.	return 0;을 while문의 닫는 중괄호 } 밖으로 이동	return 0은 "프로그램 전체 종료"이므로, 모든 반복이 끝난 뒤 마지막에 써야 함.

계산 오류

(여백 길이 문제)
	자릿수마다 여백을 더할 때 맨 앞이나 뒤의 여백이 하나 부족함.	

int total_length = 1;

(초기값을 1로 설정)
	(숫자+여백)을 세트로 묶어 더하면, 맨 앞의 시작 여백 하나가 남으므로 미리 채워줌.
